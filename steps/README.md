# STEP files overview

This folder contains all 3D STEP files for the iPolarisFinder hardware.

## RPI box (2 files)

- `iPolarisFinder-RPI_box.step` — Raspberry Pi box base.
- `iPolarisFinder-RPI_box_top.step` — Raspberry Pi box top/cover.

## Camera module (3 files)

- `iPolarisFinder-lens.step` — Lens camera PCB holder.
- `iPolarisFinder-body.step` — Long body tube to insert in the base.
- Base options (choose one):
  - `iPolarisFinder-Base_DIY_mount.step` — Base for DIY mount.
  - `iPolarisFinder-Base_EQ5_SW.step` — Base for Sky-Watcher EQ5 mount.
  - `iPolarisFinder-Base_EQ5_EXOS2.step` — Base for Bresser EXOS2 mount.
